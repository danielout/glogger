# Chat — All Messages (Global Search)

## Overview

The most feature-rich chat tab — search across all messages from all channels with advanced filtering.

## How It Works

- **Text search** — full-text search across message bodies using FTS index
- **Item name filter** — search for messages mentioning a specific item
- **"Item links only" toggle** — show only messages that contain item references
- **Channel badges** — each message shows which channel it came from
- **Paginated results** — 100 messages per page

## Use Cases

- Finding who was selling a specific item in Trade chat
- Searching for mentions of a quest or location across all channels
- Reviewing all item-linked messages to track market activity
