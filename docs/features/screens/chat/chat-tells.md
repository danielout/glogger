# Chat — Tells (Direct Messages)

## Overview

View private 1-to-1 conversations with other players.

## How It Works

- **Conversation sidebar** — lists all players you've exchanged tells with, showing message count per conversation
- **Select a conversation** to view the message thread
- **Bubble layout** — messages displayed as chat bubbles: your messages on the right, theirs on the left
- **Paginated loading** — 100 messages per page
- Item links in messages rendered via `ItemInline`
