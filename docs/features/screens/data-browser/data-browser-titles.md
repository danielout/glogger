# Data Browser — Titles

## Overview

Browse player titles with color tag rendering, scope indicators, and tooltip text.

## Search

- Text search across title text (with color tags stripped) and tooltip (computed, no debounce)
- All titles loaded on mount

## Detail View

- **Title text** — rendered with inline color from `<color=#XXXXXX>` tags
- ID
- **Tooltip text**
- **Scope badges** — Account-Wide, Soul-Wide indicators
- **Keywords** — blue badges
- **Raw JSON**

## List View

Account-wide titles show a purple "acct" badge in the search results list.
