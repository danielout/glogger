# Data Browser — Abilities

## Overview

Browse abilities organized by skill, with combat details, upgrade chains, and flag inspection.

## Search & Filters

- **Skill filter dropdown** — only shows skills that have abilities
- **Text search** — client-side within selected skill, or global search across all skills (debounced 250ms)

## Detail View

- Icon
- ID, skill, level requirement, icon ID
- **Combat details** — damage type, cooldown, target, range, mana/power cost, animation
- **Special info** — prerequisites, special_info field
- **Flags** — extracted from raw JSON (Harmless, Works Underwater, etc.)
- **Related abilities** — upgrade chains and shared cooldowns
- **Sources** — where the ability is obtained
- **PvE/PvP details** — raw JSON blocks for environment-specific data
- **Keywords**
- **Raw JSON**
