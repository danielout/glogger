# Data Browser — Skills

## Overview

Browse all game skills with descriptions, advancement hints, and related abilities.

## Search

- Text search across skill name and description (computed, no debounce)
- All skills loaded on mount

## Detail View

- Icon
- ID, icon ID, XP table reference
- **Skill details** — combat vs non-combat flag, max bonus levels, guest level cap, advancement table
- **Parents** — prerequisite skills
- **Hide when zero** flag
- **Advancement hints** — organized by level threshold
- **Related abilities** — all abilities belonging to this skill, sorted by level
- **Keywords**
- **Raw JSON**
