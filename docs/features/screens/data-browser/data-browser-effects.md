# Data Browser — Effects

## Overview

Browse game effects (buffs, debuffs, status effects) with duration and stacking details.

## Search

- Text search by effect name (debounced 250ms)
- Result count displayed when results exist

## Detail View

- Icon
- ID, display mode, icon ID
- **Description**
- **Details** — duration (formatted hours/minutes/seconds), stacking type, stacking priority, particle effect
- **Keywords** — blue badges
- **Ability Keywords** — green badges (separate from regular keywords)
- **Raw JSON**
