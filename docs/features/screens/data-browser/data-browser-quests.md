# Data Browser — Quests

## Overview

Browse quests with area filtering, sorting, and full requirement/reward breakdowns.

## Search & Filters

- **Text search** — across name, description, internal name, and area
- **Area filter dropdown** — dynamically populated from quest data
- **Cancellable filter** — yes / no / all
- **Sort options** — by name, level, or area

## Detail View

- Display name, internal name, level, area
- **Description**
- **Quest Giver Dialog** — PrefaceText in styled callout
- **Quest Info** — cancellable, reuse time, favor NPC, work order skill
- **Requirements** — with diamond bullet points
- **Objectives** — type badges + description + count
- **Rewards** — favor amount, skill/combat XP (green checkmarks), item rewards with stack sizes, named loot profile
- **Completion Dialog** — SuccessText in styled callout
- **Sources**
- **Keywords**
- **Raw JSON**
