# Widget: Notes

**ID:** `player-notes` | **Default size:** Medium | **Component:** `PlayerNotes.vue`

Simple personal checklist:
- Add new notes via text input
- Toggle checkbox completion
- Delete individual notes
- Clear all completed notes

Persisted to `localStorage` (key: `glogger-player-notes`). Intentionally not in the database — these are scratch-pad items, not game state.
